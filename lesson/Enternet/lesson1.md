# 1 лекция

**Как работает интернет**

*15.08.2025 Время занятия: `2`(с 14:00 до 16:00)*

## Основные понятия:

- **Пакет** - единица данных, передаваемая через интернет
- **Маршрутизатор** - устройство, которое перенаправляет пакеты данных между устройствами
- **IP-адрес** - уникальный идентификатор, присваиваемый устройству в сети и используемый для маршрутизации данных
  получателю
- **Доменное имя** - уникальный адрес сайта в интернете, по которому страницы ресурса доступны пользователям
- **DNS (Domain Name System)** - система доменных имён, отвечающая за преобразование доменного имени в IP-адрес
- **HTTP (HyperText Transfer Protocol)** - протокол передачи гипертекста, используемый для передачи данных между
  клиентом и сервером
- **HTTPS (HyperText Transfer Protocol Secure)** - протокол защищённой передачи данных
- **SSL (Secure Sockets Layer)/TLS (Transport Layer Security)** - протоколы, используемые для обеспечения безопасности
  связи через Интернет
- **Браузер** - программное обеспечение для просмотра веб-страниц

---

## Что такое Интернет

**Сеть** - группа компьютеров или других устройств, соединённых друг с другом. Тогда **интернет** - это сеть сетей.

Интернет - это глобальная сеть компьютеров:

- Соединённых друг с другом
- Взаимодействующих посредством стандартизированного набора протоколов
- Где каждое подключённое устройство должно иметь уникальный адрес

На высоком уровне абстракции интернет работает за счёт:

1. Соединения устройств и компьютерных систем
2. Использования определённого набора протоколов

Эти протоколы:

- Определяют порядок обмена информацией между устройствами
- Обеспечивают надёжную и безопасную передачу данных

**Ядро интернета** - глобальная сеть взаимосвязанных маршрутизаторов, которые:

- Перенаправляют трафик между устройствами и системами
- При получении данных разбивают их на пакеты
- Последовательно передают пакеты до конечного пункта назначения

Для обеспечения передачи данных используются:

- **IP** - отвечает за маршрутизацию пакетов
- **TCP** - обеспечивает надёжную передачу пакетов в правильном порядке

---

## Роль протоколов в интернете

**Протокол** - набор правил и стандартов, определяющий порядок обмена информацией между устройствами.

Преимущества стандартизированных протоколов:

- Обеспечивают бесперебойную связь между устройствами разных производителей
- Позволяют взаимодействовать разным системам (например, веб-браузеру одной компании с веб-сервером другой компании при
  использовании HTTP)

## Что такое IP-адрес и доменные имена

### IP-адрес

- Уникальный идентификатор устройства в сети
- Имеет формат: `nnn.nnn.nnn.nnn` где `nnn` - число от 0 до 255
- Пример: `192.168.1.1`

### Доменные имена

- Удобочитаемые названия для идентификации веб-сайтов
- Состоят из двух и более частей, разделённых точками
- Пример: `google.com`

### Взаимодействие DNS

1. Пользователь вводит доменное имя в браузер
2. Компьютер отправляет DNS-запрос
3. DNS-сервер возвращает соответствующий IP-адрес
4. Компьютер использует IP-адрес для подключения к ресурсу
